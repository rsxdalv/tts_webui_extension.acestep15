# Ace Step 1.5 extension for [TTS WebUI](https://github.com/rsxdalv/tts-webui).

## Installation

```bash
pip install git+https://github.com/rsxdalv/tts_webui_extension.acestep15@tts-webui
```

## Usage

1. Install the extension
2. Restart TTS Generation WebUI
3. Navigate to the Acestep15 tab

## Development

To run the extension standalone:

```bash
cd tts_webui_extension/acestep15
python main.py
```
